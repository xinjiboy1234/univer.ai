# Introduction to Univer Doc

<Callout emoji="⚠️">
  Univer Doc is now in the early stages of development, most of the features are not yet available. Please follow our [Roadmap](/guides/doc/roadmap) to stay updated on our progress.
</Callout>

## Features

### 🛠 General Capabilities

#### Editing Operations

- Undo / Redo
- Copy / Paste

#### Viewing

- High-performance rendering engine based on Canvas, supporting multiple document types
- Scroll and zoom

#### UI

- Configurable workspace, including title bar, toolbar, sidebar, status bar, context menu, etc.
- Cross-platform shortcut key support
- Shortcut key panel

### 📝 Document

#### Editing Operations

- Text Styles: bold, italic, underline, strike through, font family, font size, text color
